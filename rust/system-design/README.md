# System Design

System design exercises in Rust covering scalability patterns.

## Exercises

1. **ex1_caching** - LRU cache, TTL cache
2. **ex2_load_balancing** - Round-robin, least connections
3. **ex3_rate_limiting** - Token bucket, sliding window
4. **ex4_sharding** - Consistent hashing
5. **ex5_message_queue** - Simple queue, pub/sub
6. **ex6_designs** - URL shortener, KV store

## Testing

```bash
cd rust/system-design
cargo test
```
